# Discussed Logo Assets

Final brand assets that are safe to keep in the public repo.

## Primary assets

- `discussed-logo.svg`
  Combination mark for website headers, landing pages, and brand presentation.
- `discussed-mark-brand.svg`
  Larger standalone mark for favicon, social/avatar use, and general brand display.
- `discussed-mark-toolbar.svg`
  Badge-safe standalone mark for the browser toolbar icon.

## PNG exports

PNG exports live in `logos/final/png/`.

- `discussed-logo-512.png`
- `discussed-logo-1024.png`
- `discussed-logo-1200.png`
- `discussed-logo-2048.png`

- `discussed-mark-brand-{16,32,48,96,128,256,512}.png`
- `discussed-mark-toolbar-{16,32,48,96,128,256,512}.png`

## Project usage

- Website / landing page header:
  `logos/final/discussed-logo.svg`
- Website favicon / brand icon:
  `logos/final/discussed-mark-brand.svg`
- Extension toolbar icon:
  `public/icon/*.png` generated from `logos/final/discussed-mark-toolbar.svg`
